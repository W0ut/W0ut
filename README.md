-= W0ut =-

WHY IS IT ALL?

    building scripts for easy and automatic use of hacking wifi and a little more 

HOW TO INSTALL

    ~# cd W0ut
    ~# dos2unix install.sh
    ~# chmod +x install.sh
    ~# ./install.sh
EXAMPLE

    ~#./getpass.sh
    ~#./getpass.sh /home/wifi.cap

    ~#./hashcrack.sh
    ~#./hashcrack.sh /home/filehash.hash
